# T-0001 — Fix CPU Temperature Not Displaying on Dashboard

**Status:** [DONE]  
**Branch:** task-T-0001-fix-cpu-temp-display  
**Date:** 2026-05-28

## Problem

CPU temperature shows `--` on the dashboard even though the API returns it correctly.

## Root Cause

Data path mismatch between backend and frontend:

| | Path | Key |
|--|--|--|
| **Backend returns** | `hardware.temperature.cpu_celsius` | `cpu_celsius` |
| **Frontend reads** | `hardware.cpu.temperature_c` | `temperature_c` |

The frontend looks for temperature nested inside the `cpu` object with a `_c` suffix; the backend puts it in a separate top-level `temperature` object with a `_celsius` suffix.

## Fix

In `templates/dashboard.html`, change:
```js
const temp = hardware.cpu?.temperature_c ?? "--";
```
to:
```js
const temp = hardware.temperature?.cpu_celsius ?? "--";
```
